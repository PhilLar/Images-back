imgase-back
=====

`images-back` is the webservice to store and process images.

Installation
------------

Install `images-back` from sources, by running:

```sh
git clone https://github.com/PhilLar/Images-back.git
cd short
go install ./cmd/images-back
```

Usage
-----
You can run it:
```sh
images-back
```
